# CyberAI Toolkit

A powerful open-source cybersecurity toolkit for security professionals and enthusiasts.  
Includes password auditing, cryptographic utilities, network OSINT, and payload generation.  
Built with **Node.js** and enhanced by **artificial intelligence** for advanced analysis.

---

## ✨ Features

- 🔑 **Password Auditing**  
  Test password strength, detect weak credentials, and simulate brute-force scenarios.

- 🔒 **Cryptographic Utilities**  
  Hashing, encryption/decryption, and digital signature verification with modern algorithms.

- 🌐 **Network OSINT**  
  Gather intelligence from public sources, analyze domains, IPs, and metadata.

- 🎯 **Payload Generation**  
  Create and customize payloads for penetration testing and red-team exercises.

- 🤖 **AI-Powered Analysis**  
  Leverages machine learning models to detect anomalies, classify threats, and provide predictive insights.

## 🚀 Getting Started

### Prerequisites

### AI Integration
The toolkit uses AI models to:
- Detect suspicious patterns in network traffic
- Predict potential attack vectors
- Classify cryptographic strength
- Provide recommendations for remediation


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
